# Data Ingestion API

This is a FastAPI-based data ingestion API project with batch processing and priority support.

---

## Requirements

- Python 3.10 or higher  
- FastAPI  
- Uvicorn==0.22.0  
- Other dependencies as listed in `requirements.txt`
